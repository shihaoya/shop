## 1. 系统管理员操作流程

### 1.1 登录系统
```
访问PC端登录页面 → 输入用户名和密码 → 点击登录 → 进入管理后台
```

### 1.2 审核运营方注册申请

#### 流程步骤：
1. **查看待审核列表**
   - 登录系统后进入"运营方管理"模块
   - 点击"待审核"标签页
   - 查看所有待审核的运营方申请

2. **审核操作**
   - 点击某个申请查看详情
   - 查看申请人信息和申请描述
   - 选择审核结果：
     - **通过**：运营方账号激活，可以登录使用
     - **拒绝**：填写拒绝原因，申请状态变为已拒绝

3. **查看审核历史**
   - 切换到"已审核"标签页
   - 查看所有已通过或拒绝的申请记录

### 1.3 管理所有用户

#### 查看用户列表
```
用户管理 → 查看所有用户 → 可按角色筛选（管理员/运营方/普通用户）
```

#### 重置用户密码
```
用户管理 → 找到对应用户 → 点击"重置密码" → 
系统生成随机密码 → 记录密码并通过线下方式告知用户
```

#### 禁用/启用用户
```
用户管理 → 找到对应用户 → 点击"禁用"或"启用" → 确认操作
```

### 1.4 查看系统日志

#### 操作日志查询
```
日志管理 → 操作日志 → 设置筛选条件（时间范围、操作人、操作类型等）→ 查询
```

#### 查看日志详情
```
点击某条日志 → 查看详细操作信息（操作人、时间、IP、操作内容、结果等）
```

### 1.5 管理系统租户

#### 查看租户列表
```
租户管理 → 查看所有运营方租户 → 查看租户状态、创建时间等信息
```

#### 禁用/启用租户
```
租户管理 → 找到对应租户 → 点击"禁用"或"启用" → 确认操作
注：禁用后该运营方下所有功能不可用
```

## 2. 运营方用户操作流程

### 2.1 注册与登录

#### 注册流程
```
访问注册页面 → 选择"运营方注册" → 
填写用户名、密码 → 填写申请描述 → 提交申请 → 
等待管理员审核 → 审核通过后即可登录
```

#### 登录流程
```
访问登录页面 → 输入用户名和密码 → 点击登录 → 进入运营方管理后台
```

### 2.2 商品管理

#### 新增商品
```
商品管理 → 点击"新增商品" → 
填写商品信息：
  - 商品名称（必填）
  - 商品描述（可选）
  - 所需积分（必填，正整数）
  - 库存数量（必填，正整数）
  - 上传商品图片（可选）
  - 商品分类（可选）
→ 点击"保存" → 商品创建成功（默认下架状态）
```

#### 上架商品
```
商品管理 → 找到对应商品 → 点击"上架" → 确认操作 → 商品对用户可见
```

#### 下架商品
```
商品管理 → 找到对应商品 → 点击"下架" → 确认操作 → 用户无法看到该商品
```

#### 编辑商品
```
商品管理 → 找到对应商品 → 点击"编辑" → 
修改商品信息 → 点击"保存" → 更新成功
```

#### 删除商品（逻辑删除）
```
商品管理 → 找到对应商品 → 点击"删除" → 确认操作 → 
商品移至回收站（is_deleted=1）
```

#### 查看回收站
```
商品管理 → 点击"回收站"标签 → 查看已删除商品
```

#### 恢复商品
```
回收站 → 找到对应商品 → 点击"恢复" → 商品恢复正常状态
```

### 2.3 用户管理

#### 手动添加用户

**情况1：添加已注册用户**
```
用户管理 → 点击"添加用户" → 
输入用户名搜索 → 选择用户 → 
设置初始积分 → 点击"确认" → 
用户添加到当前运营方名下
```

**情况2：添加未注册用户**
```
用户管理 → 点击"添加用户" → 
选择"创建新用户" → 
填写用户名、密码（可设置默认密码）、昵称 → 
设置初始积分 → 点击"确认" → 
系统自动创建用户并关联到当前运营方
```

#### 批量导入用户
```
用户管理 → 点击"批量导入" → 
下载CSV模板 → 填写用户信息 → 
上传文件 → 系统校验数据 → 
确认导入 → 完成批量添加
```

#### 查看用户列表
```
用户管理 → 查看所有用户 → 
支持按用户名搜索 → 
显示用户基本信息和当前积分余额
```

#### 查看用户详情
```
用户管理 → 点击某个用户 → 
查看用户详细信息：
  - 基本信息（用户名、昵称等）
  - 当前积分余额
  - 历史兑换记录
  - 积分变动流水
```

#### 移除用户（逻辑删除）
```
用户管理 → 找到对应用户 → 点击"移除" → 确认操作 → 
用户移至回收站，但仍保留在系统中
```

#### 恢复用户
```
用户管理 → 点击"回收站"标签 → 
找到对应用户 → 点击"恢复" → 用户恢复正常状态
```

### 2.4 积分管理

#### 查看用户积分列表
```
积分管理 → 查看所有用户积分列表 → 
显示每个用户的当前积分余额 → 
支持按用户名搜索用户
```

#### 查看用户积分详情
```
积分管理 → 点击某个用户 → 
进入该用户的积分详情页 → 
查看积分历史变动记录：
  - 变动时间
  - 操作人
  - 变动类型（增加/减少/修改）
  - 变动数量
  - 变动后余额
  - 变动原因
→ 支持按时间范围、变动类型筛选
```

#### 增加积分
```
积分管理 → 找到对应用户 → 点击"增加积分" → 
输入增加数量和原因（必填）→ 
系统校验（数量必须为正整数）→ 
点击"确认" → 积分增加成功 → 
记录积分流水 → 发送通知给用户
```

#### 减少积分
```
积分管理 → 找到对应用户 → 点击"减少积分" → 
输入减少数量和原因（必填）→ 
系统校验（数量必须为正整数，不能超过当前余额）→ 
点击"确认" → 积分减少成功 → 
记录积分流水 → 发送通知给用户
```

#### 直接修改积分
```
积分管理 → 找到对应用户 → 点击"修改积分" → 
输入新的积分值和原因（必填）→ 
系统校验（新值必须为非负整数）→ 
点击"确认" → 积分修改成功 → 
记录积分流水 → 发送通知给用户
```

#### 批量调整积分
```
积分管理 → 点击"批量调整" → 
选择多个用户 → 
选择调整类型（增加/减少/修改）→ 
输入调整数量和原因（必填）→ 
系统逐个校验 → 点击"确认" → 
批量调整成功 → 记录流水 → 发送通知
```

#### 导出积分流水
```
积分管理 → 进入用户积分详情 → 
点击"导出流水" → 选择时间范围 → 
生成Excel文件 → 下载保存
```

### 2.5 订单管理

#### 查看所有订单
```
订单管理 → 查看所有订单列表 → 
显示订单编号、用户、商品、积分、状态、时间等信息 → 
支持按订单状态、时间范围筛选
```

#### 查看订单详情
```
订单管理 → 点击某个订单 → 
查看订单详细信息：
  - 订单编号
  - 用户信息
  - 商品信息
  - 消耗积分
  - 收货地址（如有）
  - 订单状态
  - 创建时间
```

#### 更新订单状态
```
订单管理 → 找到对应订单 → 点击"更新状态" → 
选择新状态（待处理/已完成/已取消）→ 
点击"确认" → 状态更新成功 → 
发送通知给用户
```

#### 导出订单
```
订单管理 → 点击"导出订单" → 
选择时间范围和状态 → 
生成Excel文件 → 下载保存
```

### 2.6 消息管理

#### 查看消息列表
```
消息中心 → 查看所有消息 → 
显示消息标题、类型、时间、是否已读 → 
未读消息标红显示
```

#### 查看消息详情
```
消息中心 → 点击某条消息 → 
查看消息详细内容 → 
系统自动标记为已读
```

#### 标记已读/未读
```
消息中心 → 选中消息 → 
点击"标记已读"或"标记未读"
```

#### 删除消息（逻辑删除）
```
消息中心 → 选中消息 → 点击"删除" → 
消息移至回收站
```

## 3. 普通用户操作流程

### 3.1 注册与登录

#### 注册流程
```
访问注册页面 → 选择"普通用户注册" → 
填写用户名、密码、昵称 → 
同意用户协议 → 点击"注册" → 
注册成功，自动登录
```

#### 登录流程
```
访问登录页面 → 输入用户名和密码 → 
点击"登录" → 进入用户个人中心
```

### 3.2 申请加入运营方

#### 浏览运营方列表
```
首页/运营方列表 → 查看所有运营方 → 
显示运营方名称、简介等信息
```

#### 申请加入运营方
```
运营方列表 → 点击某个运营方 → 
查看运营方详情 → 点击"申请加入" → 
确认申请 → 提交申请 → 
等待运营方审核
```

#### 查看申请状态
```
个人中心 → 我的申请 → 
查看所有申请记录及状态：
  - 待审核
  - 已通过
  - 已拒绝
```

### 3.3 切换运营方视角

#### 选择运营方
```
个人中心 → 我的运营方 → 
显示所有已加入的运营方列表 → 
点击某个运营方 → 切换到该运营方视角
```

#### 查看当前运营方信息
```
切换后，页面显示：
  - 当前运营方名称
  - 在该运营方的积分余额
  - 该运营方的商品列表
```

### 3.4 浏览和兑换商品

#### 浏览商品列表
```
商品列表 → 查看当前运营方的所有上架商品 → 
显示商品图片、名称、所需积分、库存等信息 → 
支持按积分范围、分类筛选
```

#### 查看商品详情
```
商品列表 → 点击某个商品 → 
查看商品详细信息：
  - 商品名称和描述
  - 商品图片
  - 所需积分
  - 剩余库存
  - 其他说明
```

#### 兑换商品

**步骤1：确认兑换**
```
商品详情 → 点击"立即兑换" → 
系统检查积分余额是否充足 → 
如果不足，提示"积分不足" → 
如果充足，进入确认页面
```

**步骤2：填写收货信息（可选）**
```
确认页面 → 填写收货地址（可选）→ 
如不填写，可线下联系运营方 → 
点击"确认兑换"
```

**步骤3：完成兑换**
```
系统扣除相应积分 → 
生成订单记录 → 
显示兑换成功提示 → 
发送通知给运营方 → 
跳转到订单详情页
```

### 3.5 查看个人积分

#### 查看积分余额
```
个人中心 → 积分信息 → 
显示当前运营方下的积分余额
```

#### 查看积分流水
```
个人中心 → 积分流水 →

  - 变动时间
  - 变动类型（增加/减少/兑换消耗）

### 3.6 查看订单记录

```
个人中心 → 我的订单 → 
查看所有订单记录 → 
显示订单编号、商品、积分、状态、时间等信息 → 
支持按订单状态筛选
```

#### 查看订单详情
```
我的订单 → 点击某个订单 → 
查看订单详细信息：
  - 订单编号
  - 商品信息
  - 消耗积分
  - 收货地址（如有）
  - 订单状态
  - 创建时间
```

### 3.7 个人中心

#### 查看个人信息
```
个人中心 → 个人信息 → 
显示用户名、昵称等基本信息
```

#### 修改个人信息
```
个人中心 → 个人信息 → 点击"编辑" →

## 5. 异常处理流程

### 5.1 登录失败
```
输入错误密码 → 系统提示"用户名或密码错误" → 
连续失败5次 → 账号锁定30分钟 → 
等待解锁或联系管理员
```

### 5.2 积分不足
```
兑换商品时 → 系统检查积分 → 
积分不足 → 提示"积分不足，当前余额XX，需要XX" → 
引导用户联系运营方增加积分
```

### 5.3 库存不足
```
兑换商品时 → 系统检查库存 → 
库存不足 → 提示"商品库存不足" → 
引导用户选择其他商品
```

### 5.4 网络异常
```
操作过程中 → 网络断开 → 
提示"网络连接异常，请检查网络" → 
自动重试或手动刷新页面
```

### 5.5 权限不足
```
访问无权限页面 → 系统检测权限 → 
提示"无权访问该页面" → 
自动跳转到有权限的页面
```

## 6. 操作注意事项

### 6.1 运营方注意事项
1. **积分调整**：每次调整积分都必须填写原因，便于追溯
2. **商品管理**：下架商品前确认没有进行中的订单
3. **用户管理**：移除用户前确认该用户没有未完成的订单
4. **订单处理**：及时处理用户订单，更新订单状态
5. **数据安全**：妥善保管账号密码，定期修改密码

### 6.2 普通用户注意事项
1. **积分兑换**：兑换前确认积分余额充足
2. **收货地址**：如需邮寄，务必填写准确的收货地址
3. **订单查询**：及时查看订单状态，如有疑问联系运营方
4. **账号安全**：保管好账号密码，不要泄露给他人
5. **申请加入**：申请加入运营方后，耐心等待审核

### 6.3 管理员注意事项
1. **审核严谨**：认真审核运营方申请，确保合规
2. **密码重置**：重置密码后及时线下告知用户
3. **日志监控**：定期查看系统日志，发现异常及时处理
4. **数据备份**：定期备份数据库，防止数据丢失
5. **系统维护**：定期检查系统运行状态，确保稳定
